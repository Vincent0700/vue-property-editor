## Installation

### npm 

 ```bash
npm i -S vue-property-editor
```

### CDN

```html
<!-- import CSS -->
<link rel="stylesheet" href="https://unpkg.com/vue-property-editor/lib/vue-property-editor.css">
<!-- import JavaScript -->
<script src="https://unpkg.com/vue-property-editor/lib/vue-property-editor.umd.js"></script>
```
